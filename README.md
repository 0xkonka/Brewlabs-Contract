# Brewlabs Staking
Smart Contracts for Brewlabs Staking

## Pools
### Brewlabs Pool A

Stake Brewlabs Earn Brewlabs + reflections (compound and harvest options)

- Duration 365 Days
- Liquidity 2%
- 2% deposit fee
    - 2% send to "Address A"
- 4% withdrawal fee 
    - 3% send to "Address A"
    - 1% sent to burn address
